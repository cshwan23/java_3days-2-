<문제 1>
num3의 값은?

		int num1 = 5;
		int num2 = 2;
		double num3 = num1 / num2;
		
		System.out.println("numb3 => " + num3 );
		
<정답>
		 num3에 2.0이 저장되는 이유?	 
     
<이유> 
(자료형 우선순위)

=> num1/num2 연산결과인 int형 2가 double형 변수에 저장되므로 
2 -> 2.0으로 형 변환이 된다.

=> 데이터 끼리 연산시 자료형이 서로 틀리면 한쪽이 다른 쪽에 맞춰 자료형이 변환되고, 데이터를 변수에 저장할 경우에도 데이터가 변수의 자료형으로 변환된다.
